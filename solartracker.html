Solar Tracker Project

  • Received second place in a project where the goal was to create a device to collect more energy from the sun than it expended and maximize efficiency
  • The image on the left is of the prototype created using laser cut wood, two servo motors, four solar panels, four LDRs, and an Arduino. I wrote the code for the system using C++.
  • The solar tracker I created is dual axis, so it takes into account not only the movement of the sun and Earth throughout the day, but also throughout the year.
  • The tracker moves into the most optimal position by taking the inputs from the 4 LDRs separated by walls, and moving to a position where the input values are all equal.
